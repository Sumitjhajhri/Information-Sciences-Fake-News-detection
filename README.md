# Research Project: Analysis of Machine Learning Models

## 📌 Overview
This project contains multiple experiments (RQ1–RQ7) to evaluate machine learning models under different conditions such as leakage, imbalance, robustness, and scalability.

## 📂 Code Files
- RQ1_leakage_aware_evaluation.ipynb – evaluates data leakage issues
- RQ2_tabular_vs_graph_learning.ipynb – compares tabular vs graph learning
- RQ3_prevalence_shift_and_imbalance.ipynb – studies imbalance problems
- RQ4_hybrid_fusion.ipynb – hybrid model approach
- RQ5_robustness_to_degraded_features.ipynb – tests robustness
- RQ6_performance_efficiency_scalability.ipynb – efficiency & scalability
- RQ7_interpretability_and_actionability.ipynb – model interpretability

## 📊 Dataset
The dataset used is provided in the **Data** folder.

## 📈 Outputs
All generated figures, graphs, and tables are available in the **outputs** folder.

## 📁 Additional Files
- Network Architecture (PDF/PPT)
- Workflow Figure

## ▶️ How to Run
Open the Jupyter notebooks and run each file step by step.
